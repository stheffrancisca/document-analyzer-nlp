# 📄 Document Analyzer NLP

Machine Learning-ready document analysis pipeline built with Python and NLP techniques.

---

## 🚀 Features

- Text preprocessing (tokenization, stopword removal)
- TF-IDF feature engineering
- Document classification using Naive Bayes
- Model evaluation with accuracy metric
- Modular ML project structure

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- NLTK
- TF-IDF
- Naive Bayes
